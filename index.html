Introduction

JavaScript is a lightweight, interpreted programming language with first-class functions.

It is best known as the scripting language for Web pages.

It is an essential part of the web development stack, along with HTML and CSS.

You can write JavaScript in a script tag or in a separate `.js` file.

Modern JavaScript supports ES6+ syntax and modules.

console.log("Hello, world!");
  • Dynamically typed
  • Prototype-based
  • Event-driven
Variables

Variables are containers for storing data values.

You can declare them using `let`, `const`, or `var`.

`let` and `const` are block-scoped.

let name = "Najoa"; const PI = 3.14;

`var` is function-scoped and should be avoided in modern code.

  • let — reassignable
  • const — constant
  • var — legacy keyword
Functions

Functions are blocks of code designed to perform a task.

They are defined with the `function` keyword or as arrow functions.

function greet(name) { return "Hello " + name; } const add = (a, b) => a + b;

Functions can be passed as values, making JavaScript a functional language.

  • Function Declaration
  • Function Expression
  • Arrow Function
Control Flow

Control flow refers to the order in which individual statements are executed.

Conditional statements are used to perform different actions based on conditions.

if (x > 10) { console.log("Big"); } for (let i = 0; i < 5; i++) { console.log(i); }

Loops such as `for`, `while`, and `do...while` are used for iteration.

  • if / else
  • switch
  • for / while
DOM Manipulation

The DOM (Document Object Model) represents the structure of a web page.

You can use JavaScript to dynamically update content, styles, and structure.

document.getElementById("demo").innerHTML = "Changed!"; document.querySelector(".btn").addEventListener("click", () => alert("Clicked!"));

DOM manipulation is central to creating interactive web apps.

  • getElementById
  • querySelector
  • addEventListener
